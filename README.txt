South West England Football - Netlify + Admin Website

UPLOAD TO NETLIFY:
1. Go to Netlify
2. Add new site
3. Deploy manually
4. Drag this ZIP in

TO USE ADMIN EDITOR:
Decap CMS requires GitHub + Netlify Identity/Git Gateway.
After upload:
1. In Netlify, connect this site to a GitHub repository
2. Go to Site configuration > Identity > Enable Identity
3. Go to Identity > Services > Enable Git Gateway
4. Invite yourself as a user
5. Visit /admin on your website

If you do not set up GitHub/Identity, the website still works, but the admin editor will not save changes.
